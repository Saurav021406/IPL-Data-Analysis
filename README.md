# IPL Data Analysis & Visualization

This repository contains a Data Science project based on IPL cricket data analysis using Python.

## Project Overview

The project analyzes IPL datasets to extract insights about:

- Team performance
- Player statistics
- Toss impact
- Season trends
- Venue analysis

## Files Included

- code/IPL_Data_Analysis.ipynb – Python notebook with full analysis
- report/Project_Report.docx – Detailed project report
- presentation/Project_Presentation.pdf – Final presentation

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## How to Use

1. Download the repository  
2. Open the Jupyter Notebook  
3. Run the cells to view analysis and visualizations
